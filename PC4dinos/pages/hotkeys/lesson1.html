Тема 1. Копировать, Вырезать, Вставить

  • ctrl + c - Сохраняет файл или другой элемент в буфер обмена для дальнейшего копирования.
  • ctrl + x - Сохраняет файл или другой элемент в буфер обмена для дальнейшего переноса в другое место.
  • ctrl + v - Выводит в текущее положение (каталог, строку) файл из буфера обмена и применяет предыдущее действие (копироване, вырезание).
